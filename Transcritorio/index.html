About / Sobre

Transcritorio is a desktop application for transcribing research interviews entirely on your local machine. It was built for the InfoCitizen project (ERC, 2022 Brazilian Census) but works with any audio/video interview.

All processing happens locally — no audio is sent to the cloud. This is critical for research involving sensitive data (informed consent forms, vulnerable populations, confidential interviews).

O Transcritorio é um aplicativo desktop para transcrição de entrevistas de pesquisa inteiramente no computador local. Nenhum áudio é enviado para a nuvem — essencial para pesquisas com dados sensíveis.

Features

WhisperX ASR

State-of-the-art speech recognition using OpenAI's Whisper large-v3 model with word-level timestamps and forced alignment.

Speaker Diarization

Automatic speaker identification using pyannote.audio. Distinguishes interviewer from interviewee with exclusive speaker assignment.

GPU Acceleration

CUDA-accelerated processing on NVIDIA GPUs. A 1-hour interview transcribes in ~5 minutes on an RTX 4060. CPU fallback available.

Review Studio

Built-in GUI for reviewing and editing transcriptions. Audio player synced with text, waveform visualization, and keyboard shortcuts.

Multiple Formats

Export to Markdown, DOCX, JSON, SRT subtitles, VTT, and TSV. Auditable canonical JSON preserves all metadata.

100% Local

No internet required after initial setup. Audio never leaves your machine. Models cached locally. DPAPI-encrypted tokens.

Installation / Instalação

  1. Download both files from the GitHub Release page (scroll down to "Assets"):
    • Transcritorio-0.1.0-Setup.exe (2.7 MB — the installer)
    • Transcritorio-0.1.0-Setup-1.bin (1.9 GB — the data)
    Both files are required. The installer will not work without the .bin file. Keep both in the same folder (e.g., Downloads).
    Ambos os arquivos são necessários. O instalador não funciona sem o arquivo .bin. Mantenha os dois na mesma pasta.
  2. Run the installer — double-click Transcritorio-0.1.0-Setup.exe. No admin rights required — installs per-user.
  3. First launch: The app will ask for your Hugging Face token (free account at huggingface.co) to download ASR and diarization models (~3 GB, one-time only). The token is stored securely via Windows DPAPI and never sent anywhere.
  4. Verify installation: Open a terminal and run transcritorio-cli.exe self-test to check CUDA, FFmpeg, and dependencies.

Quick Start / Primeiro Uso

  1. Open Transcritorio from the Start Menu or desktop shortcut.
  2. Create or open a project — point to a folder containing your audio/video files. The app creates a Transcricoes/ subfolder for all outputs.
  3. Build the manifest — the app scans for media files and lists them. Select which interviews to transcribe.
  4. Transcribe — click the Transcribe button. The pipeline runs: audio preparation → ASR (WhisperX) → diarization (pyannote) → rendering.
  5. Review & edit — use the Review Studio to play audio synced with the transcript, correct errors, merge/split turns, and reassign speakers.
  6. Export — save final transcriptions as Markdown, DOCX, or other formats.

Privacy & Security

Transcritorio was designed for research with sensitive data. Key privacy guarantees:

  • 100% local processing — no audio, video, or transcript ever leaves your machine.
  • No cloud dependency — after initial model download, the app works fully offline.
  • Secure token storage — Hugging Face tokens are encrypted via Windows DPAPI (never stored in plain text).
  • Original files untouched — the app writes all outputs to a Transcricoes/ folder; original audio/video/TCLE files are read-only.
  • No telemetry — no analytics, no tracking, no phone-home behavior.

FAQ / Troubleshooting

Do I need an NVIDIA GPU?

No, but it's strongly recommended. With a GPU (e.g., RTX 4060, 8 GB VRAM), a 1-hour interview transcribes in ~5 minutes. On CPU, it may take 30-60 minutes.

What is the Hugging Face token for?

The diarization model (pyannote) requires accepting a license on Hugging Face. The token authenticates the one-time model download. It is stored locally via DPAPI and never sent elsewhere.

Can I use it for languages other than Portuguese?

Yes. WhisperX supports 90+ languages. Change asr_language in the config file (e.g., en for English, es for Spanish).

CUDA not detected / CPU fallback

Run transcritorio-cli.exe self-test to diagnose. Common causes: outdated NVIDIA drivers, GPU without CUDA support, or insufficient VRAM. The app works on CPU as fallback.

How do I verify my installation?

Run transcritorio-cli.exe self-test in a terminal. It checks CUDA availability, FFmpeg, and all dependencies.

Where are the models stored?

In %LOCALAPPDATA%\Transcritorio\models\huggingface\. They take about 3 GB and are downloaded once.

Is macOS / Linux supported?

Not yet. The standalone installer is Windows-only for now. The Python package can be installed from source on any platform.
